# Planteamiento del problema - Calculadora

Realice cálculos matemáticos simples y avanzados utilizando la aplicación de calculadora.

1. Los cálculos básicos incluyen sumas, restas, multiplicaciones y divisiones.
- Suma de números enteros y decimales
- Resta de números enteros y decimales
- Multiplicación de números enteros y decimales
- División de números enteros (Asegúrese de que un número no se divide por cero. Maneje la situación apropiadamente).​
- Módulo de números enteros y decimales

2. Las operaciones científicas incluyen funciones exponenciales y raíz cuadrada
- Calcular el valor entero mayor o igual más próximo a un número dado.
- Calcular el número del valor de un número al múltiplo o al entero inferior más próximo de la cifra significativa especificada
- Raíz cuadrada de un número no negativo
- Calcular una expresión exponencial​

*** Nota: ***​
- Los números tomados como entrada deben ser no negativos. Si el número ingresado es 0 o negativo, genere un error y maneje la situación.
- Manejar todos los escenarios erróneos apropiadamente.
​
Diseñe una aplicación de calculadora que le ayude a realizar cálculos matemáticos 
y científicos.

# Instrucciones

1. Descargue y descomprima el código.
2. Ejecute el comando `npm install` para instalar las dependencias.
3. Abra el código en VSCode para desarrollar la solución.
4. Escriba el código en los archivos **.js** presentes en la carpeta **src**
5. Ejecute los scripts de prueba disponibles en **src/test** dando el comando `npm run test` en la terminal para probar localmente.
6. Refactorice la solución para garantizar que se superen todos los casos de prueba.
7. Comprima el código de la solución con el mismo nombre que el nombre de la tarea.
8. Cargue la solución comprimida para enviarla.
